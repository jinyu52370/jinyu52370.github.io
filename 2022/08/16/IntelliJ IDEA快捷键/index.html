IntelliJ IDEA快捷键

发表于2022-08-16|更新于2022-08-16|编程JavaIDE
|字数总计:77|阅读时长:1分钟|阅读量:
操作 快捷键
书签 Ctrl + F11
返回 Ctrl + Alt + ←
展示类结构 Ctrl + F12
在资源管理器中打开 Ctrl + Alt + F12
最近编辑文件列表 Ctrl + E
定位行 Ctrl + G
转去定义 Ctrl + B / Ctrl + 鼠标左键
查找引用 Alt + F7
定位下一个错误 F2
重命名 Shift + F6
